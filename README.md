# Ecommerce-webapp

# 🛍️ Meesho Clone Web App

A full-stack e-commerce web application inspired by Meesho, built using modern technologies like React, Node.js, Express, and MongoDB. This project replicates core functionalities of an online shopping platform including product browsing, authentication, cart management, and order processing.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based Login & Signup)
- 🛒 Add to Cart & Cart Management
- 📦 Product Listing with Category Filtering
- 💳 Order Placement Workflow
- 📱 Fully Responsive UI (Mobile + Desktop)
- ⚡ Optimized Backend APIs for fast performance
- 🔄 Seamless Frontend & Backend Integration

---

## 🧰 Tech Stack

### 💻 Frontend
- React.js

- HTML5 & CSS3
- Tailwind CSS / Bootstrap

### ⚙️ Backend

### 🗄️ Database
